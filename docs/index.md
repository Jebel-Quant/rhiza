# Rhiza Documentation

Welcome to the official Rhiza documentation.

## What is Rhiza?

Rhiza provides reusable configuration templates for modern Python projects. It helps you maintain consistent tooling, CI/CD pipelines, and development workflows across multiple repositories.

## Quick Start

```bash
# Install dependencies
make install

# Run tests
make test

# Build documentation
make book

# See all available targets
make help
```
